Cartão de Visitas

Este código HTML e CSS cria um cartão de visita virtual com animação de virada. O cartão apresenta informações sobre e exibe foto, detalhes de contato e links para suas redes sociais (GitHub, LinkedIn, Instagram) com ícones interativos. A animação "flip" permite alternar entre a frente e o verso do cartão ao passar o mouse sobre ele.

Estante de Livros

Este é um site de estante de livros que apresenta informações sobre vários livros, incluindo capas, sinopses e detalhes como data de publicação, autor, idioma original, gêneros, número de páginas e editora. Super Mário

O projeto "Super MarioJS" é um jogo baseado em HTML, CSS e JavaScript que recria a atmosfera do clássico jogo Super Mario. O JavaScript controla o pulo do Mario ao pressionar uma tecla e verifica colisões com o cano, interrompendo o jogo caso ocorra uma colisão. O design visa capturar a essência do jogo original com elementos icônicos e animações. Uma lembrança de infância.
